# 简擎后端 v0.1 发布说明

发布日期：2026-03-06  
版本：`v0.1.0`

## 本版摘要

`v0.1` 聚焦“管理内核最小闭环”，完成认证、RBAC、审计三大核心能力，并补齐基础工程化门禁与开源文档。

## 新增与完善

- 认证与会话
  - 登录 / 登出 / 当前用户接口
  - JWT 鉴权链路
  - Redis 会话校验与 token TTL 配置
- 权限与系统管理
  - 用户、角色、菜单基础能力
  - 用户分配角色、角色分配菜单
  - 接口方法统一约束为 `GET` / `POST`
- 审计能力
  - 登录日志、操作日志分页查询
  - 支持关键字/状态/登录方式筛选
- 缓存与一致性
  - 热点数据缓存
  - 延迟双删策略保障 DB 与缓存一致性
- 工程化与文档
  - CI：编译 + 测试
  - Checkstyle 静态检查
  - README、贡献规范、API 示例文档

## 兼容与迁移说明

- 本版本依赖：`JDK 17+`、`MySQL 8+`、`Redis 6+`
- 若从旧版本演进，请按 `sql/patch` 中脚本完成增量修复。

## 已知限制

- 当前自动化测试仍以基础命令验证为主，业务回归测试需在 v0.2 继续补齐。
- 中间件（ES/Nacos/RocketMQ）以适配层预留为主，未默认强依赖接入。

## 下一步（v0.2 方向）

- 增强数据权限与更细粒度权限模型
- 补齐核心业务自动化回归测试
- 提供中间件接入示例与发布流程增强
