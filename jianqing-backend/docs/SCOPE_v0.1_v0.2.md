# 范围说明：v0.1 与 v0.2

## v0.1（当前范围）

### 已包含

- 认证与会话
  - 登录、登出、当前用户信息
  - JWT + Redis 会话双校验
  - token TTL 可配置
- 系统管理
  - 用户、角色、菜单基础 CRUD
  - 用户分配角色、角色分配菜单
- 审计能力
  - 登录日志、操作日志
  - 分页与筛选查询（关键字/状态/登录方式）
- 工程化
  - CI（编译、测试）
  - Checkstyle 静态检查
  - 开源基础文档（README/CONTRIBUTING/API_EXAMPLES）

### 不包含（明确边界）

- 复杂工作流引擎与审批流
- 完整租户体系与细粒度数据权限
- 默认接入 Elasticsearch / Nacos / RocketMQ 的生产级配置
- 完整自动化测试体系（当前以编译、基础测试与静态检查为主）

## v0.2（规划方向）

### 目标增强

- 权限模型增强
  - 数据权限与更细粒度按钮/资源权限策略
- 集成能力增强
  - ES / Nacos / RocketMQ 的可选接入示例与文档
- 质量能力增强
  - 核心业务回归测试（认证、RBAC、审计）
  - 更完整的发布流程与校验脚本
- 管理体验增强
  - 结合前端持续优化筛选、空态反馈与交互一致性

### 里程碑建议

1. `v0.2-alpha`：完成权限模型增强与关键回归测试
2. `v0.2-beta`：补齐中间件接入示例与文档
3. `v0.2.0`：发布稳定版并完善发布流程自动化
